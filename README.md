## What is this
MEAN stack boilerplate - with Angular 2 
(will be updated to Angular 4)

## Check 
https://www.npmjs.com/package/mean-stack-boilerplate

## How to run/install
+ `npm install`
+ `node server.js`
+ `npm start`
+ Navigate to `localhost:3000/`
